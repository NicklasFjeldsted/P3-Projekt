Utils

Rendering

Canvas

Properties

  • "_context" is a private member of each canvas instance. The CanvasRendering2DContext allows us to manipulate the canvas.

  • "_element" is a private member of each canvas instance. This referes to the HTML element that the canvas is within the DOM.

Methods

  • CalculateLocalPointFrom(globalPoint: Vector2)

    Takes in a Vector2.

    Returns either a Vector2 or null.

    This method calculates the local canvas mouse position from the global DOM mouse position.

  • SetStyle(style: Partial)

    Takes in a Partial of type CSSStyleDeclarion.

    Doesn't return anything.

    This method takes a TypeScript type that contains some CSS properties and converts them to actual CSS and then applies it to the canvas.

  • FillRect(start: Vector2, size: Vector2, color: Color)

    Takes in a start position of type Vector2, a size of type Vector2, and an optional color of type Color.

    Doesn't return anything.

    This method draws a rect on the canvas.

  • ExtendedRect(start: Vector2, size: Vector2, shape: Shape)

    Takes in a start position of type Vector2, a size of type Vector2, and a shape of type Shape.

    Doesn't return anything.

    This method is a better version of the FillRect method, as this one can draw custom shapes.

  • StrokeRect(start: Vector2, size: Vector2, color: Color)

    Takes in a start position of type Vector2, a size of type Vector2, and an optional color of type Color.

    Doesn't return anything.

    This method draws a stroke rect.

  • FillCircle(center: Vector2, radius: number, color: Color)

    Takes in a center position of type Vector2, a radius of type Number, and a color of type Color.

    Doesn't return anything.

    This method is for drawing a circle to the canvas.

  • ClearRect(start: Vector2, size: Vector2)

    Takes in a start position of type Vector2, and a size of type Vector2.

    Doesn't return anything.

    This method clears a specific area of the canvas.

  • ClearCanvas()

    Doesn't take anything.

    Doesn't return anything.

    This method clears the entire canvas.

  • DrawText(text: string, position: Vector2, fontSize: number, maxWidth?: number, fit: boolean, color?: Color, shadow: boolean, blur: number, shadowColor: Color, shadowOffset: Vector2)

    Takes in a text of type String, a position of type Vector2, a font size of type Number, a max width of type Number, a fit of type boolean, a blur of type Number, a shadow color of type Color, and a shadow offset of type Vector2

    Doesn't return anything.

    This method draws text to the canvas.

  • MeasureText(text: string)

    Takes in a text of type String.

    Returns a number.

    This method measures a text of type string and returns the width in pixels.

  • DrawImage(source: string, start: Vector2, size: Vector2, color?: Color)

    Takes in a source/path of type string, a start position of type Vector2, a size of type Vector2, and an optional color of type Color.

    Doesn't return anything.

    This method draws an image to the canvas.

  • RecolorImage(image: HTMLImageElement, width: number, height: number, position: Vector2, newColor: Color)

    Takes in an image of type HTMLImageElement, a width of type number, a height of type number, a position of type Vector2, and a new color of type Color.

    Doesn't return anything.

    This method takes an image and recolors it, going through each pixel in the image and applying a new color over it.

  • IsInverted(width: number, height: number)

    Takes in a width of type number, and a height of type number.

    Returns a boolean.

    This method check if a rendered image is inverted, I.E. it folds over itself.

Canvas Layer

Canvas Layer is a static class that creates and retrieves canvas' to create a feel of depth.

Properties

  • "_layers" is an array of canvas' that have been created.

  • "Layers" is a public getter for "_layers".

Methods

  • GetLayer(index: number)

    Takes in an index of type number.

    Returns a canvas.

    This method gets the canvas that has the index of the input parameter.

  • InitCanvas(style: Partial)

    Takes in a Partial type of CSSStyleDeclarion.

    Returns a canvas.

    This method creates a new canvas and sets its starting CSS styling.

GameObject

The class GameObject inherits from Entity.

All GameObjects have either a Transform or a RectTransform component attached to them.

Properties

  • "isActive" is a boolean that controls whether or not the GameObject is active.

  • "_awoken" is a private boolean that is true if the GameObject has run its awake method this runtime.

  • "_game" is a Game that is set when the GameObject is instantiated.

  • "_parent" is the parent GameObject of this GameObject.

  • "_children" is an array of GameObject that have their parent set to this GameObject.

  • "gameObjectName" is the name of this GameObject.

  • "_components" is a private array of all the components on this GameObject.

  • "transform" is a public getter that returns the Transform component on this GameObject.

Methods

  • Child(index: number)

    Takes in an index of type number.

    Returns a GameObject

    This method takes in the index of a child set to this "_children" array and returns the child at the given index.

  • SetParent(newParent: GameObject)

    Takes in a new parent of type GameObject.

    Doesn't return anything.

    This methods sets this GameObject's parent to the given GameObject new parent parameter.

  • AddComponent(component: IComponent, index?: number)

    Takes in a component that implements the IComponent interface, it also takes in an optional index of type number.

    Returns a GameObject.

    This method adds the given component to this GameObject, if the index parameter has a value, the component will replace the component at the given index in the "_components" array.

  • GetComponent< C extends IComponent >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IComponent.

    Returns a component of type C.

    This method searches for the constructor type that was given and returns the component if it is on the GameObject.

  • RemoveComponent< C extends IComponent >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IComponent.

    Doesn't return anything.

    This methods remove the type of C from this GameObject "_components" array.

  • HasComponent< C extends IComponent >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IComponent.

    Returns a boolean.

    Checks if this GameObject has the type of C component in its "_components" array.

Entity

Entities implements IAwake, IStart and IUpdate

Properties

  • "entityId" is a type of string, this is a global unique id.

  • "_features" is an array of classes that implements the IFeature interface.

Methods

  • AddFeature(feature: IFeature)

    Takes in a feature that is a class that implements the IFeature interface.

    Returns an Entity.

    This method adds the feature parameter to this entity's "_features" array.

  • GetFeature< C extends IFeature >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IFeature.

    Returns a feature of type C.

    This method searches this entitys "_features" array for the given constructor and returns it.

  • RemoveFeature< C extends IFeature >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IFeature.

    Doesn't return anything.

    This method searches for the given constructor in this entitys "_features" array and removes it.

  • HasFeature< C extends IFeature >(constr: constr< C >)

    Takes in a constructor of type C which is an abstract constructor that implements the IFeature.

    Returns a boolean.

    This method checks this entitys "_features" array for the given constructor and returns whether or not it contains it.
